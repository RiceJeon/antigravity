   인 캠페인은 자동으로 중단되며 문제 해결 시 재개됩니다.

Description

1. 보안 검증 (Secondary Auth)
- 광고주 계정에 직접 접속하여 민감 데이터 열람 시, 최고관리자 본인 확인을 위해 비밀번호를 재입력받음

2. 시각적 포커스
- 배경 화면(PRJ-02-06)에 Blur(블러) 처리를 적용하여 모달에 시선을 집중시킴

3. 입력 및 액션
- 비밀번호 입력 후 '확인' 시 해당 GEO 메인 네비 > 대시보드로 접속 처리
- '취소' 또는 '닫기(X)' 시 모달을 닫고 이전 상세 관리 화면으로 복귀

Description

1. 요금제 선택 모달
- 최고관리자가 광고주의 요금제를 강제로 변경하기 위해 노출되는 플랜 선택 화면

2. 현재 플랜 상태
- 광고주가 현재 이용 중인 플랜(PRO)은 '사용 중'으로 표시되며 버튼이 비활성화됨

3. 플랜 활성화
- 나머지 모든 플랜은 선택 가능한 상태로 노출되며, 클릭 시 사유 입력 단계로 이동

요금제 변경 사유 입력
플랜을 'STANDARD'으로 변경하시겠습니까? 사유를 입력해 주세요.
✔ 요금제가 변경되었습니다. 익월부터 반영될 예정입니다.

Description

1. 변경 사유 입력
- 강제 변경 처리의 투명성을 위해 운영팀의 변경 사유를 필수로 입력받음

2. 토스트 알림 (Toast Message)
- 확인 클릭 시 요금제 변경 성공 메시지가 화면 하단에 일시적으로 노출됨
- 문구: "요금제가 변경되었습니다. 익월부터 반영될 예정입니다."

3. 후속 조치
- 토스트 노출 후 모달이 닫히며, 이전 상세 관리 화면(PRJ-02-06)으로 복귀함
